工具：enca
[cc lang="bash"]
wget http://dl.cihar.com/enca/enca-1.13.tar.gz
tar xzvf enca-1.13.tar.gz
./configure
make && make install
[/cc]
里面有enca和enconv两个工具可以查看和转换编码
[cc lang="bash"]
[root@gateway /]#enca -L none test.php
Universal transformation format 8 bits; UTF-8
