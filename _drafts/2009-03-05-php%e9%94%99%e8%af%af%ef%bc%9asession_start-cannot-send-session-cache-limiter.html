有关session_start() Cannot send session cache limiter 这个错误，有时程序本身是没有错误的，至少我的问题是如此。在WINDOWS下面写程序，有的类似模块，为了方便，就复制一份，结果导致文件的变化。可能大家都不注意，只有问题出现了才解决。

解决办法：新建一个文件，将以前的文件内容复制一份就可以了。其实也可以在LINUX下面测试，用VI打开这个文件应该多了点东西，删除也是可以的。
